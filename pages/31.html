Use OPAM

OPAM is a package manager for OCaml, which can be used to install Coq and many of its packages. The general documentation can be found on the OPAM website. For example, to install the version 1.2.2 of OPAM from the sources on a Unix system:

curl -L https://github.com/ocaml/opam/archive/1.2.2.tar.gz |tar -xz
cd opam-1.2.2
./configure
make lib-ext
make
sudo make install

To add the repository for the Coq packages:

opam repo add coq-released https://coq.inria.fr/opam/released

To install Coq:

opam install -j4 -v coq

To install a package:

opam install coq:that-super-package

To add the repository of development packages (use it at your own risks):

opam repo add coq-extra-dev https://coq.inria.fr/opam/extra-dev

To add the repository of development versions of Coq (use it at your own risks):

opam repo add coq-core-dev https://coq.inria.fr/opam/core-dev
